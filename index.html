03 — Project

PROJECT

One project, documented end to end — problem, build, and outcome.

Live & deployed

Fish Feed Recommendation System

A web-based platform that recommends appropriate feed types and quantities to fish farmers based on pond and species data. The goal was to replace manual guesswork with a simple, data-backed recommendation a farmer can actually use in the field.

Open live platform ↗ Ask about the build

Problem

  • Farmers lacked a simple way to size feed to pond & species
  • Manual estimation led to waste and inconsistent yields

Approach

  • Structured farmer inputs into a recommendation model
  • Built a lightweight, farmer-facing web interface
  • Deployed as a live, publicly accessible platform
